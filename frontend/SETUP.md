### 1. Environment Setup
Create a `.env` file in the `frontend` directory:

```bash
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 2. Install & Run
```bash
cd frontend
npm install
npm run dev
```

### 3. Supabase Setup Required
- Create Supabase project
- Run the SQL schema from the main README
- Create storage buckets: `resumes`, `offers`, `certificates`, `terms`
- Set up RLS policies
